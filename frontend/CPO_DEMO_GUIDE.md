## ✅ HARD GATE 체크리스트

### API 계약 테스트
- [ ] `GET /api/current` → providers 배열 반환
- [ ] `GET /api/trends?range=24h` → points 배열 반환
- [ ] `GET /api/trends?range=7d` → points 배열 반환
- [ ] `GET /api/trends?range=30d` → points 배열 반환
- [ ] `GET /api/providers` → providers 배열 반환
- [ ] `GET /healthz` → status ("ok"|"degraded"|"error")
- [ ] 잘못된 range 파라미터 → 400 에러

### UI 상태 구분
- [ ] **Empty**: 데이터 없음 (파란 안내 박스)
- [ ] **Error**: 수집 실패 (빨간 에러 박스 + last_error)
- [ ] **Stale**: 오래됨 (노란 경고 박스 + 마지막 수집 시각)
- [ ] **Warning**: 사용량 80%+ (노란/빨간 진행바)
- [ ] **Normal**: 정상 (초록 진행바)

### 5 초 룰 (Risk Assessment)
- [ ] Risk Banner 항상 visible (데이터 있을 때)
- [ ] 4 개 카운터: 위험/주의/오래됨/정상
- [ ] 색상 코딩: 빨강/노랑/노랑/초록
- [ ] 한 줄 평가: "🔴 위험 - 즉시 조치 필요" 등
- [ ] 자동 업데이트 (30 초 간격)

### Provider 카드 정보
- [ ] Provider 명
- [ ] 사용량 / 한도 / 잔여량
- [ ] 사용률 (%)
- [ ] 진행바 (색상 코딩)
- [ ] 마지막 수집 시각 (last_success_at)
- [ ] 다음 리셋 시각 (reset_at)
- [ ] 상태 배지 (정상/주의/에러/오래됨)

### 반응형 디자인
- [ ] 데스크톱: 3+ 컬럼 그리드
- [ ] 태블릿: 2 컬럼
- [ ] 모바일: 1 컬럼
- [ ] 차트: 자동 크기 조절
